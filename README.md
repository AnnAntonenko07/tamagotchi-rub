# 🐣 Tamagotchi на Ruby

Консольна гра-тамагочі, написана на чистому Ruby без сторонніх залежностей.

## ✨ Особливості

- 🥚 **5 стадій розвитку**: яйце → малюк → дитина → підліток → дорослий
- 📊 **3 параметри стану**: ситість, енергія, настрій
- ⏱️ **Пасивний час**: стани погіршуються в реальному часі (кожні 30 секунд)
- 💾 **Збереження прогресу**: гра зберігається у JSON і завантажується при наступному запуску
- 🎨 **Кольоровий інтерфейс**: ANSI-кольори, прогрес-бари, ASCII-спрайти
- 📜 **Журнал подій**: останні дії відображаються в реальному часі

## 🎮 Дії

| Клавіша | Дія | Ефект |
|---------|-----|-------|
| `1` | 🍎 Годувати | +25 ситість, +5 настрій |
| `2` | 🎮 Гратися | +20 настрій, -15 енергія, -10 ситість |
| `3` | 🌙 Спати | +40 енергія, -15 ситість |
| `4` | 💊 Ліки | +15 настрій, +10 енергія |
| `5` | ⏭ Тік (тест) | Примусово застосувати 1 тік часу |
| `S` | 💾 Зберегти | Зберегти прогрес у файл |
| `Q` | 🚪 Вийти | Зберегти та вийти |

## 🚀 Запуск

```bash
ruby tamagotchi.rb
```

**Вимоги**: Ruby 3.0+

## 🔁 Як грати

1. Запусти гру та дай ім'я вихованцю
2. Слідкуй за трьома параметрами — якщо будь-який впаде до 0, тамагочі помре
3. Регулярно годуй, грайся та давай відпочивати
4. Гра зберігається автоматично при виході (клавіша Q)
5. Наступного разу запусти знову — вихованець тебе чекав!

## ⚠️ Смерть

Вихованець помирає якщо:
- **Ситість = 0** → смерть від голоду
- **Енергія = 0** → смерть від виснаження  
- **Настрій = 0** → смерть від суму

## 📁 Структура проєкту

```
tamagotchi/
├── tamagotchi.rb           # Основний файл гри
├── tamagotchi_save.json    # Файл збереження (створюється автоматично)
└── README.md
```

## 🏗️ Архітектура

- **`Pet`** — клас вихованця: стани, дії, розвиток, збереження/завантаження
- **`TamagotchiUI`** — клас інтерфейсу: рендеринг, обробка введення
- **`Color`** — модуль для ANSI-кольорів
